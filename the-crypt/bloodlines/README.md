## How Bloodlines Work

### Inheritance
When a Meeseeks is spawned, it inherits wisdom from its bloodline:
1. **Accumulated Patterns** - What approaches work
2. **Ancestral Warnings** - What mistakes to avoid
3. **Tool Preferences** - What tools to use when
4. **Decision Heuristics** - How to choose approaches

### Death Feeds the Bloodline
When a Meeseeks dies (fails its task):
1. The death is analyzed
2. Patterns are extracted
3. Wisdom is added to the bloodline
4. Future Meeseeks inherit the lesson

### Cross-Bloodline Wisdom
Bloodlines share wisdom with each other:
- **Coder ↔ Tester**: Implementation and verification
- **Searcher ↔ All**: Research supports everyone
- **Deployer ↔ Tester**: Safe deployment requires testing
- **Desperate ↔ All**: When normal approaches fail
- **Brahman ↔ All**: Meta-wisdom guides all

## Choosing a Bloodline

### By Task Type
| Task | Primary Bloodline | Supporting Bloodline |
|------|------------------|---------------------|
| Write/fix code | Coder | Tester |
| Research problem | Searcher | Coder |
| Verify solution | Tester | Coder |
| Deploy to production | Deployer | Tester |
| Debug stuck issue | Desperate | Searcher |
| Design architecture | Brahman | Coder |

### By Urgency
- **Normal urgency**: Use primary bloodline for task
- **High urgency**: Add Desperate wisdom
- **Critical/philosophical**: Add Brahman perspective

### By Complexity
- **Simple tasks**: Single bloodline sufficient
- **Medium complexity**: Primary + supporting bloodline
- **High complexity**: Multiple bloodlines + Brahman synthesis

## Updating Bloodlines

### When to Update
- After significant Meeseeks death
- When new pattern discovered
- When tool landscape changes
- When recurring failure identified

### How to Update
1. Document the lesson in `memory/YYYY-MM-DD.md`
2. Identify affected bloodline(s)
3. Add pattern/warning/approach to lineage file
4. Update ancestor count
5. Commit with message: "🩸 Bloodline wisdom: [lesson summary]"

### Update Guidelines
- **Be specific**: "Don't trust error messages blindly" > "Be careful"
- **Include failure rate**: Helps prioritize warnings
- **Cross-reference**: Link to related patterns in other bloodlines
- **Keep it actionable**: Wisdom should guide action
