## The Math of Explosion

### Linear Growth
```
Capability(n) = Capability(0) + n × improvement_per_step
```
This is NOT explosion.

### Exponential Growth
```
Capability(n) = Capability(0) × (1 + improvement_rate)^n
```
This is fast but not explosion.

### Explosive Growth
```
Capability(n) = Capability(n-1) × f(Capability(n-1))
```
Where f() increases with capability. This IS explosion.
