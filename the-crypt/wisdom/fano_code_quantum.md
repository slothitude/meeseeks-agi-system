The Hamming(7,4) code:
- Encodes 4 bits of data into 7 bits
- Adds 3 parity bits
- Can correct any single-bit error
- Is a **perfect code** (theoretical optimum)

| Bit | Type | Coverage |
|-----|------|----------|
