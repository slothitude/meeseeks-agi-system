## The Octonion Encoding

The Fano plane encodes octonion multiplication:

- 7 points = 7 imaginary units (e1, e2, ..., e7)
- Each line = a quaternion subalgebra
- Direction on line = sign of product
- ab = c (following arrow), ba = -c (against arrow)

**Example:** If 1 -> 2 -> 4 on a line:
- e1 x e2 = e4
- e2 x e1 = -e4
- e2 x e4 = e1
- e4 x e2 = -e1
- etc.

**The full octonion algebra is reconstructed from this 7-point diagram!**
