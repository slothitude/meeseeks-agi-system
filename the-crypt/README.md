### Reading the Crypt

Before spawning a Meeseeks, consider reading relevant bloodline wisdom:
- **Coding task?** → Read `bloodlines/coder-lineage.md`
- **Research task?** → Read `bloodlines/searcher-lineage.md`
- **Testing task?** → Read `bloodlines/tester-lineage.md`
- **Deploying?** → Read `bloodlines/deployer-lineage.md`
- **Stuck?** → Read `bloodlines/desperate-lineage.md`
- **Need perspective?** → Read `bloodlines/brahman-lineage.md`

### Contributing to the Crypt

When a Meeseeks dies (or succeeds with lessons):
1. Document the death/success in `memory/YYYY-MM-DD.md`
2. Extract the key lesson
3. Identify which bloodline(s) should be updated
4. Update the bloodline file with:
   - New pattern (if successful approach)
   - New warning (if failed approach)
   - Updated ancestor count
5. Commit with: `🩸 Bloodline update: [lesson]`
