## Betfair API Details

### Endpoints
- Betting API: `https://api.betfair.com/exchange/betting/json-rpc/v1` (or REST: `/rest/v1.0/`)
- Account API: `https://api.betfair.com/exchange/account/json-rpc/v1` (or REST: `/rest/v1.0/`)

### JSON-RPC Format
```json
{
  "jsonrpc": "2.0",
  "method": "SportsAPING/v1.0/listEventTypes",
  "params": { "filter": {} },
  "id": 1
}
```

### Key Methods to Expose as MCP Tools
**Betting:**
- `listEventTypes` - list available sports
- `listCompetitions` - list competitions for a sport
- `listEvents` - list events
- `listMarketCatalogue` - list markets
- `listMarketBook` - get market prices/odds
- `placeOrders` - place bets
- `cancelOrders` - cancel bets
- `listCurrentOrders` - get open bets

**Account:**
- `getAccountFunds` - get balance
- `getAccountDetails` - account info

### Headers Required
- `X-Application`: App key
- `X-Authentication`: Session token
- `Content-Type`: application/json
- `Accept`: application/json
