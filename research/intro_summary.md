# Summary: Hacking - The Art of Exploitation (2nd Edition)

**Author:** Jon Erickson

## Key Points

- **Hands-on exploitation approach**: Unlike most security books that just show how to use existing exploits, this book teaches how to *develop* them from scratch using real code and detailed explanations.

- **Comprehensive technical foundation**: Covers the full stack needed for exploitation — C programming, x86 assembly, memory segmentation, buffer overflows, format string vulnerabilities, networking, shellcode, and cryptology.

- **Practical attack techniques**: Includes hands-on topics like network sniffing, TCP/IP hijacking, denial of service attacks, and countermeasures — all with working code examples.
